# Bacardi - Crea un Mojito 🚀

Puesta en práctica de conocimiento de HTML,SCSS y JavaScript. Realizando una pequeña api donde puedes contruir tu propio mojito. Utilizando las funciones de Drag and Drop, se logra crear el efecto de contrucción del mojito desde 0.

Está adaptada para diferentes tamaños de pantallas.

**únicamente se puede probar en ordenador.

## Tecnología 💻
- HTML
- CSS
- Java
- SCSS
